# AI Email Agent

## Overview

AI Email Agent is an intelligent email automation system that fetches emails, summarizes their content using Generative AI, and delivers concise daily digests through messaging platforms. The project aims to reduce information overload and improve productivity.

## Features

* Automated Email Fetching
* AI-powered Email Summarization
* Daily Digest Generation
* Telegram Notifications
* Workflow Automation
* Secure API Integration

## Tech Stack

* Node.js
* Google Gemini AI
* Gmail API
* Telegram Bot API
* n8n Workflow Automation

## Workflow

Gmail → AI Processing → Summary Generation → Telegram Delivery

## Project Highlights

* Integrated multiple APIs
* Automated repetitive communication tasks
* Generated concise summaries using AI
* Designed scalable workflow architecture
